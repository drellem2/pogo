# VS Code Integration

> **Status**: Planned. Not yet implemented.

A VS Code extension that integrates with the pogo daemon for project discovery and code search.

## Planned Features

- **Command palette** project switcher
- **Search results panel** for cross-project code search
- **Tree view** for listing known projects
- **Status bar** indicator showing current project and indexing status

## Requirements (expected)

- VS Code 1.75+
- pogo server running (`pogo server start`)

Check the [GitHub repository](https://github.com/drellem2/pogo) for updates on availability.
